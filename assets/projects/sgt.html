Student Grade Table
Live   Github
Student Grade Table is an academic content management system that is able to perform the 4 CRUD operations on student information: create, read, update, and delete. The total averages of all students is also calculated and displayed for the user to see.
This application was developed using JavaScript/jQuery, HTML, and Bootstrap3 on the frontend for the design and responsiveness. The backend was developed using PHP with a MySQL database. AJAX calls were made to the server to ensure endpoints were established to enable CRUD operations.
